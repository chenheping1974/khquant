### 在A股真正有效的因子（实证验证）

```
因子           A股实证结果            vs 美股

市值(Size)     ✅ 有效                一致 (小盘溢价)
价值(Value)    ✅ 有效(用E/P)         一致 (但必须用E/P而非B/P!)
盈利(Profit)   ✅ 有效(ROE)           一致
投资(Inv)      ⚠️ 反向! (-0.35%/月)   相反! (A股: 激进投资=高回报)
动量(Mom)      ⚠️ 弱于美股            一致方向, 但幅度小
情绪(Sentiment) ✅ 有效(0.83%/月)      特有 (散户主导市场)
低波动(LowVol) ⚠️ 有争议              弱于美股
```

### Fama-French 五因子在A股的失效

```
FF 传统因子      A股月均溢价    T值      结论
HML (价值)       0.20%        0.90     ❌ 不显著!
RMW (盈利)       0.18%        0.89     ❌ 不显著!
CMA (投资)       -0.35%       -2.43    ❌ 显著为负! (方向反了)

→ Fama-French 原版模型直接搬到A股是错的
→ 必须用 LSY 中国版: 剔除最小30%股票 + E/P替代B/P
```

## 二、幻方量化的因子框架 (4/3/2/1)

### 来源构成

```
价量因子    40%  ████████████████████
基本面因子  30%  ███████████████
另类数据    20%  ██████████
其他        10%  █████
```

### 各类别详细内容

**价量因子 (40%)**
- 自适应动量 (不仅是简单收益, 而是经波动率调整的趋势强度)
- 波动率聚集效应 (低波动后→低波动, 高波动后→高波动)
- 流动性溢价 (Amihud非流动性)
- 资金流因子 (大单净流入/流出, 委托队列失衡)
- 截面动量 (横截面比较, 不是时间序列)

**基本面因子 (30%)**
- 价值: PE/PB分位数 (行业内比较, 不是全市场绝对值)
- 质量: ROE, 毛利率, 现金流质量
- 成长: 营收/利润增速, 研发投入
- 预期外盈利修正 (SUE = Standardized Unexpected Earnings)

**另类数据 (20%)**
- 卫星遥感 (港口/工厂/仓储实时监测)
- 搜索引擎热度 + 社交媒体情绪
- 供应链物流 (GPS轨迹, 运价)
- 产业链跨品种传导

**其他 (10%)**
- 行业轮动信号
- 宏观周期 (库存周期/信贷周期)
- 事件驱动 (分红, 回购, 定增)

### 幻方的关键方法论

1. **因子不是静态的** — 通过遗传算法动态筛选, 月度淘汰衰减超15%的因子
2. **深度学习发现非线性关系** — Transformer/LSTM捕捉因子间交互
3. **行业中性化** — 因子在行业内比较, 不是全市场绝对值
4. **三层架构** — 底层因子库 → 中层策略工厂(RL) → 顶层风险预算(CVaR)
5. **SHAP可解释** — 每个预测可追溯到具体因子贡献

## 五、原则

1. **宁缺毋滥** — 每个因子必须有论文支撑或幻方同款方法论
2. **大道至简** — 因子数量不追求多, 追求每个因子独立且有效
3. **实事求是** — 回测说话, 不行就砍
4. **渐进式** — 先跑通Phase1, 再加Phase2, 不可跳过
